# MouseMap

MouseMap 是一个 macOS 菜单栏鼠标按键映射工具，用来把鼠标侧键或额外按键映射成系统级操作。

项目使用 SwiftUI 编写，直接通过 `swiftc` 构建，不依赖 Xcode 工程文件。

## 功能

- 学习鼠标额外按键并添加映射
- 将鼠标按键映射为桌面切换、调度中心、应用程序窗口、显示桌面等动作
- 菜单栏常驻运行，可快速启用或禁用映射
- 支持开机自启
- 支持配置持久化保存

## 当前内置动作

- 切换到左侧桌面
- 切换到右侧桌面
- 调度中心
- 应用程序窗口
- 显示桌面
- 禁用

## 使用方式

1. 启动应用
2. 首次运行时授予“辅助功能”权限
3. 点击菜单栏中的 `MouseMap`
4. 打开“配置...”
5. 点击“添加按键映射”
6. 按下需要学习的鼠标按键
7. 为该按键选择目标动作

## 权限要求

应用需要 macOS“辅助功能”权限才能拦截鼠标按键并触发映射。

- 首次启动或重新安装到新路径后，若权限缺失，应用会自动弹出配置窗口并触发授权请求
- 如果应用已启动但按键没有反应，优先检查“系统设置 > 隐私与安全性 > 辅助功能”里是否已勾选 `MouseMap`

## 构建

```bash
make          # 编译到 build/MouseMap.app
make run      # 编译并启动
make clean    # 清理构建产物
```

## 项目结构

```text
├── App/          # 应用入口与菜单栏窗口管理
├── Models/       # 映射动作与配置模型
├── Services/     # 权限、事件监听、配置保存、按键模拟
├── ViewModels/   # 配置界面状态管理
├── Views/        # 配置界面与权限状态界面
└── Resources/    # 应用资源与 Info.plist
```
